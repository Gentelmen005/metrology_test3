dF/dr2 = i*(-2*r1 + 2*r2)/r3 = -0.367500000000000
dF/dr3 = -i*(r1 - r2)**2/r3**2 = -3.75156250000000
dF/du0 = -1 = -1.00000000000000

Общая погрешность: 0.48707430442395355
